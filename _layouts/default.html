Description

Dracula is a prototype RTS game developed in Unity that explores systems such as hierarchical state-based AI, managing many game agents, dynamic UI handling, and more.

Custom shaders are used for the reflective crystal textures and barrier textures, and lighting is handled using Unity tools.

All scripts, shaders, and models were created by Adam Khaddaj and Leo McCarthy-Kennedy

Ambient music created by Adam Khaddaj.

3D animations are implemented using Mixamo.
